Access DB in container:
docker exec -it db_db_1 psql -U xccup_user xccup_db

Working with timezones in psql:
SHOW TIMEZONE;
SELECT * FROM pg_timezone_names; 
SET TIMEZONE='Europe/Berlin';

HowTo Connect PgAdmin to Postgres:
Add New Server -> Connection -> Hostname: <<name_of_postgres_container>> Port: 5432
Localhost as address doesn't work because PgAdmin runs in his own host!
